红黑树

简介

红黑树（Red-Black Tree）是一种自平衡的二叉搜索树，它在插入、删除等操作时通过一系列的旋转和颜色调整来保持树的平衡，从而确保树的高度始终保持在较小的范围内，保证了较快的查找、插入和删除操作的性能。

五大性质

  1. 每个节点要么是红色，要么是黑色。
  2. 根节点是黑色的。
  3. 每个叶子节点（NIL节点，空节点）是黑色的。
  4. 如果一个节点是红色的，则它的两个子节点都是黑色的。
  5. 对于每个节点，从该节点到其子孙叶子节点的所有路径上包含相同数目的黑色节点。

关键特性

  1. 从根节点到任何一个叶子节点的路径长度相同，保证了树的平衡性。
  2. 最长路径不超过最短路径的两倍，保证了树的高度始终在对数范围内，使得插入、删除和查找操作的时间复杂度为O(logN)。

应用

  1. 在C++的STL中，红黑树被用作实现map和set等关联容器。
  2. 在数据库系统中，红黑树被用作索引结构，加速数据的检索。
  3. 在操作系统中，红黑树被用来管理进程的调度顺序、文件系统的文件块分配等。
  4. 在路由算法中，红黑树被用来构建路由表，实现高效的路由查找。

红黑树的增删改查

红黑树的构建（插入节点和调整）

  1. 将新节点插入红黑树中，按照二叉搜索树的规则找到合适的插入位置。新节点默认为红色。
  2. 检查插入节点的父节点的颜色：
    • 如果父节点是黑色，插入操作结束，红黑树的性质不受影响。
    • 如果父节点是红色，进行进一步的调整。
  3. 判断插入节点的叔节点（父节点的兄弟节点）的颜色：
    • 如果叔节点是红色，表示父节点和叔节点都是红色，违反了红黑树性质4。此时，将父节点和叔节点的颜色设为黑色，将祖父节点设为红色，然后将当前节点指向祖父节点，并继续向上进行调整。
    • 如果叔节点是黑色或缺失（叶子节点被视为黑色），进行相应的旋转操作。
  4. 进行旋转操作：
    • 左旋转：如果插入节点是其父节点的右子节点，进行左旋转。通过左旋转，将插入节点变为左子节点。
    • 右旋转：如果插入节点是其父节点的左子节点，进行右旋转。通过右旋转，将插入节点变为右子节点。
  5. 调整颜色：
    • 左旋转后，交换插入节点和其父节点的颜色，使得父节点变为红色，插入节点变为黑色。
    • 右旋转后，交换插入节点和其父节点的颜色，使得父节点变为红色，插入节点变为黑色。
  6. 如果插入节点的祖父节点存在且为红色，则重复步骤2~5，直到满足红黑树的所有性质。
  7. 将根节点设为黑色，确保性质2。

红黑树节点的删除调整：

  • 情况1：删除节点是根节点，直接将根节点设为黑色。
  • 情况2：删除节点的兄弟节点是红色，将兄弟节点设为黑色，父节点设为红色，以父节点为支点进行左旋转或右旋转，转换为情况3、情况4或情况5。
  • 情况3：删除节点的兄弟节点是黑色，且兄弟节点的两个子节点都是黑色或缺失。将兄弟节点设为红色，以父节点为当前节点进行进一步调整。
  • 情况4：删除节点的兄弟节点是黑色，兄弟节点的左子节点是红色，右子节点是黑色或缺失。以兄弟节点为支点进行右旋转，交换兄弟节点和兄弟节点的左子节点的颜色，转换为情况5。
  • 情况5：删除节点的兄弟节点是黑色，兄弟节点的右子节点是红色。将父节点的颜色赋给兄弟节点，父节点设为黑色，兄弟节点的右子节点设为黑色，以父节点为支点进行左旋转，删除操作完成。

删除调整完成后，红黑树的性质得到恢复。

红黑树的查找

与二叉搜索树类似，通过比较目标值与当前节点的值，沿着树向下搜索直到找到目标值或者到达叶子节点为止。下面是红黑树的查找过程：

  1. 从根节点开始，比较目标值与当前节点的值：
    • 如果目标值等于当前节点的值，说明找到了目标节点，返回当前节点。
    • 如果目标值小于当前节点的值，则在当前节点的左子树中继续查找。
    • 如果目标值大于当前节点的值，则在当前节点的右子树中继续查找。
  2. 重复上述步骤，直到找到目标节点或者到达叶子节点（即当前节点为空）为止。
    • 如果到达叶子节点仍然没有找到目标节点，则说明目标节点不存在，返回空。

时间空间复杂度

  1. 查找操作：
    • 时间复杂度：O(log N)，其中 N 是红黑树中节点的数量。由于红黑树是一棵自平衡的二叉搜索树，每次查找操作都可以将搜索范围缩小一半，因此平均情况下的查找时间复杂度为对数级别。
    • 空间复杂度：O(1)，查找操作只需要使用常数级别的额外空间。
  2. 插入操作：
    • 时间复杂度：O(log N)，平均情况下的插入时间复杂度为对数级别。插入操作需要找到合适的位置来插入新节点，并进行一系列的旋转和颜色调整来维持红黑树的平衡性。
    • 空间复杂度：O(1)，插入操作只需要使用常数级别的额外空间。
  3. 删除操作：
    • 时间复杂度：O(log N)，平均情况下的删除时间复杂度为对数级别。删除操作需要找到待删除节点，并进行旋转和颜色调整来维护红黑树的平衡性。
    • 空间复杂度：O(1)，删除操作只需要使用常数级别的额外空间。

enum Color { RED, BLACK };

struct Node {
int data;
Color color;
Node* left;
Node* right;
Node* parent;

explicit Node(int val) : data(val), color(RED), left(nullptr), right(nullptr), parent(nullptr) {}

};

class RedBlackTree {
private:
Node* root;

// 辅助函数
Node* grandparent(Node* node) {
    if (node != nullptr && node->parent != nullptr) {
        return node->parent->parent;
    }
    return nullptr;
}

Node* sibling(Node* node) {
    if (node != nullptr && node->parent != nullptr) {
        return (node == node->parent->left) ? node->parent->right : node->parent->left;
    }
    return nullptr;
}

Node* uncle(Node* node) {
    Node* grandparent_node = grandparent(node);
    if (grandparent_node == nullptr) {
        return nullptr;
    }
    return (node->parent == grandparent_node->left) ? grandparent_node->right : grandparent_node->left;
}

void rotate_left(Node* node) {
    Node* right_child = node->right;
    if (right_child != nullptr) {
        node->right = right_child->left;
        if (right_child->left != nullptr) {
            right_child->left->parent = node;
        }
        right_child->parent = node->parent;
        if (node->parent == nullptr) {
            root = right_child;
        } else if (node == node->parent->left) {
            node->parent->left = right_child;
        } else {
            node->parent->right = right_child;
        }
        right_child->left = node;
        node->parent = right_child;
    }
}

void rotate_right(Node* node) {
    Node* left_child = node->left;
    if (left_child != nullptr) {
        node->left = left_child->right;
        if (left_child->right != nullptr) {
            left_child->right->parent = node;
        }
        left_child->parent = node->parent;
        if (node->parent == nullptr) {
            root = left_child;
        } else if (node == node->parent->left) {
            node->parent->left = left_child;
        } else {
            node->parent->right = left_child;
        }
        left_child->right = node;
        node->parent = left_child;
    }
}

void insert_fixup(Node* node) {
    while (node != root && node->parent != nullptr && node->parent->color == RED) {
        Node* uncle_node = uncle(node);
        Node* grandparent_node = grandparent(node);

        if (uncle_node != nullptr && uncle_node->color == RED) {
            node->parent->color = BLACK;
            uncle_node->color = BLACK;
            grandparent_node->color = RED;
            node = grandparent_node;
        } else {
            if (node->parent == grandparent_node->left) {
                if (node == node->parent->right) {
                    node = node->parent;
                    rotate_left(node);
                }
                node->parent->color = BLACK;
                grandparent_node->color = RED;
                rotate_right(grandparent_node);
            } else {
                if (node == node->parent->left) {
                    node = node->parent;
                    rotate_right(node);
                }
                node->parent->color = BLACK;
                grandparent_node->color = RED;
                rotate_left(grandparent_node);
            }
        }
    }
    root->color = BLACK;
}

void remove_fixup(Node* node) {
    while (node != root && node->color == BLACK) {
        if (node == node->parent->left) {
            Node* sibling_node = sibling(node);
            if (sibling_node != nullptr && sibling_node->color == RED) {
                sibling_node->color = BLACK;
                node->parent->color = RED;
                rotate_left(node->parent);
                sibling_node = sibling(node);
            }
            if (sibling_node != nullptr && sibling_node->left->color == BLACK && sibling_node->right->color == BLACK) {
                sibling_node->color = RED;
                node = node->parent;
            } else {
                if (sibling_node->right->color == BLACK) {
                    sibling_node->left->color = BLACK;
                    sibling_node->color = RED;
                    rotate_right(sibling_node);
                    sibling_node = sibling(node);
                }
                sibling_node->color = node->parent->color;
                node->parent->color = BLACK;
                sibling_node->right->color = BLACK;
                rotate_left(node->parent);
                node = root;
            }
        } else {
            Node* sibling_node = sibling(node);
            if (sibling_node != nullptr && sibling_node->color == RED) {
                sibling_node->color = BLACK;
                node->parent->color = RED;
                rotate_right(node->parent);
                sibling_node = sibling(node);
            }
            if (sibling_node != nullptr && sibling_node->right->color == BLACK && sibling_node->left->color == BLACK) {
                sibling_node->color = RED;
                node = node->parent;
            } else {
                if (sibling_node->left->color == BLACK) {
                    sibling_node->right->color = BLACK;
                    sibling_node->color = RED;
                    rotate_left(sibling_node);
                    sibling_node = sibling(node);
                }
                sibling_node->color = node->parent->color;
                node->parent->color = BLACK;
                sibling_node->left->color = BLACK;
                rotate_right(node->parent);
                node = root;
            }
        }
    }
    node->color = BLACK;
}

void insert(Node* node, int val) {
    Node* new_node = new Node(val);
    Node* current = nullptr;
    Node* parent = nullptr;

    while (node != nullptr) {
        current = node;
        if (val < node->data) {
            node = node->left;
        } else {
            node = node->right;
        }
    }

    new_node->parent = current;
    if (current == nullptr) {
        root = new_node;
    } else if (val < current->data) {
        current->left = new_node;
    } else {
        current->right = new_node;
    }

    insert_fixup(new_node);
}

void remove(Node* node, int val) {
    Node* delete_node = node;
    while (delete_node != nullptr) {
        if (val == delete_node->data) {
            break;
        }
        if (val < delete_node->data) {
            delete_node = delete_node->left;
        } else {
            delete_node = delete_node->right;
        }
    }

    if (delete_node == nullptr) {
        std::cout << "Node not found." << std::endl;
        return;
    }

    Node* child = nullptr;
    if (delete_node->left == nullptr || delete_node->right == nullptr) {
        child = delete_node;
    } else {
        child = delete_node->right;
        while (child->left != nullptr) {
            child = child->left;
        }
    }

    Node* replacement = nullptr;
    if (child->left != nullptr) {
        replacement = child->left;
    } else {
        replacement = child->right;
    }

    if (replacement != nullptr) {
        replacement->parent = child->parent;
    }

    if (child->parent == nullptr) {
        root = replacement;
    } else if (child == child->parent->left) {
        child->parent->left = replacement;
    } else {
        child->parent->right = replacement;
    }

    if (child != delete_node) {
        delete_node->data = child->data;
    }

    if (child->color == BLACK) {
        remove_fixup(replacement);
    }

    delete child;
}

void destroy(Node* node) {
    if (node != nullptr) {
        destroy(node->left);
        destroy(node->right);
        delete node;
    }
}

void print(Node* node) {
    if (node != nullptr) {
        print(node->left);
        std::cout << node->data << " ";
        print(node->right);
    }
}

public:
RedBlackTree() : root(nullptr) {}

void insert(int val) {
    insert(root, val);
}

void remove(int val) {
    remove(root, val);
}

void print() {
    print(root);
    std::cout << std::endl;
}

~RedBlackTree() {
    destroy(root);
}

};

                right_child->left->parent = node;

            right_child->parent = node->parent;
            if (node->parent == nullptr) {
                root = right_child;
            } else if (node == node->parent->left) {
                node->parent->left = right_child;
            } else {
                node->parent->right = right_child;
            }
            right_child->left = node;
            node->parent = right_child;
        }
    }

    void rotate_right(Node* node) {
        Node* left_child = node->left;
        if (left_child != nullptr) {
            node->left = left_child->right;
            if (left_child->right != nullptr) {
                left_child->right->parent = node;
            }
            left_child->parent = node->parent;
            if (node->parent == nullptr) {
                root = left_child;
            } else if (node == node->parent->left) {
                node->parent->left = left_child;
            } else {
                node->parent->right = left_child;

            node->parent = left_child;

                uncle_node->color = BLACK;
                grandparent_node->color = RED;
                node = grandparent_node;

                    if (node == node->parent->right) {
                        node = node->parent;
                        rotate_left(node);
                    }
                    node->parent->color = BLACK;
                    grandparent_node->color = RED;
                    rotate_right(grandparent_node);
                } else {
                    if (node == node->parent->left) {
                        node = node->parent;
                        rotate_right(node);
                    }
                    node->parent->color = BLACK;
                    grandparent_node->color = RED;
                    rotate_left(grandparent_node);
                }
            }
        }
        root->color = BLACK;
    }

        while (node != root && node->color == BLACK) {

                Node* sibling_node = sibling(node);
                if (sibling_node != nullptr && sibling_node->color == RED) {
                    sibling_node->color = BLACK;
                    node->parent->color = RED;
                    rotate_left(node->parent);
                    sibling_node = sibling(node);
                }
                if (sibling_node != nullptr && sibling_node->left->color == BLACK && sibling_node->right->color == BLACK) {
                    sibling_node->color = RED;
                    node = node->parent;
                } else {
                    if (sibling_node->right->color == BLACK) {
                        sibling_node->left->color = BLACK;
                        sibling_node->color = RED;
                        rotate_right(sibling_node);
                        sibling_node = sibling(node);
                    }
                    sibling_node->color = node->parent->color;
                    node->parent->color = BLACK;
                    sibling_node->right->color = BLACK;
                    rotate_left(node->parent);
                    node = root;
                }
            } else {
                Node* sibling_node = sibling(node);
                if (sibling_node != nullptr && sibling_node->color == RED) {
                    sibling_node->color = BLACK;
                    node->parent->color = RED;
                    rotate_right(node->parent);
                    sibling_node = sibling(node);
                }
                if (sibling_node != nullptr && sibling_node->right->color == BLACK && sibling_node->left->color == BLACK) {
                    sibling_node->color = RED;
                    node = node->parent;
                } else {
                    if (sibling_node->left->color == BLACK) {
                        sibling_node->right->color = BLACK;
                        sibling_node->color = RED;
                        rotate_left(sibling_node);
                        sibling_node = sibling(node);
                    }
                    sibling_node->color = node->parent->color;
                    node->parent->color = BLACK;
                    sibling_node->left->color = BLACK;
                    rotate_right(node->parent);
                    node = root;
                }
            }
        }
        node->color = BLACK;
    }

        }

        if (delete_node == nullptr) {
            std::cout << "Node not found." << std::endl;
            return;
        }

        Node* child = nullptr;
        if (delete_node->left == nullptr || delete_node->right == nullptr) {
            child = delete_node;
        } else {
            child = delete_node->right;
            while (child->left != nullptr) {
                child = child->left;
            }
        }

        Node* replacement = nullptr;
        if (child->left != nullptr) {
            replacement = child->left;
        } else {
            replacement = child->right;
        }

        if (replacement != nullptr) {
            replacement->parent = child->parent;
        }

        if (child->parent == nullptr) {
            root = replacement;
        } else if (child == child->parent->left) {
            child->parent->left = replacement;
        } else {
            child->parent->right = replacement;
        }

        if (child != delete_node) {
            delete_node->data = child->data;
        }

        if (child->color == BLACK) {
            remove_fixup(replacement);
        }

        delete child;
    }

    void destroy(Node* node) {
        if (node != nullptr) {
            destroy(node->left);
            destroy(node->right);
            delete node;
        }
    }

    void print(Node* node) {
        if (node != nullptr) {
            print(node->left);
            std::cout << node->data << " ";
            print(node->right);
        }
    }

public:
    RedBlackTree() : root(nullptr) {}

    void insert(int val) {
        insert(root, val);
    }

    void remove(int val) {
        remove(root, val);
    }

    void print() {
        print(root);
        std::cout << std::endl;
    }

    ~RedBlackTree() {
        destroy(root);
    }
};

红黑树的优化

红黑树的优化技巧：

  1. 优化节点表示：可以使用位运算或压缩存储方式来减小每个节点的内存占用。例如，可以使用位字段来表示节点的颜色和其他状态，以节省空间。
  2. 左倾红黑树（Left-Leaning Red-Black Tree）：这是一种变种的红黑树，它通过将红色链接限制为左链接，简化了插入和删除操作的实现。左倾红黑树具有与标准红黑树相同的性质，但其实现更加简单且易于理解。
  3. 节点缓存：对于频繁的插入和删除操作，可以维护一个节点缓存，以避免频繁的内存分配和释放。通过重复使用已经分配的节点，可以减少动态内存分配的开销。
  4. 批量操作：如果需要执行多个插入或删除操作，可以将它们合并为批量操作，以减少树的重构次数。在执行批量操作之后，再进行一次整体的平衡调整，可以提高性能。
  5. 局部性优化：在遍历红黑树时，可以利用局部性原理尽量保持节点在缓存中的连续访问，以提高遍历的效率。例如，可以使用中序遍历的改进版本，遍历左子树时将右子树的节点提前加载到缓存中。
  6. 惰性删除：对于需要频繁删除节点的场景，可以考虑使用惰性删除策略。即标记要删除的节点，而不是立即进行删除操作。在后续的插入或删除操作中，可以一起处理标记为删除的节点，从而减少平衡调整的次数。
  7. 局部调整：在插入或删除操作时，尽量只调整必要的节点，以减少平衡调整的范围。只有在遇到真正的平衡问题时，才进行相应的调整，避免过度调整整个树。
